# React Form Builder

## Description

- This is a google form clone created with ReactJs.
- [docs.md](./docs/index.md)

## Installation

NOTE: Node.js version 16 required to run.

```bash
git clone {GIT_URL}
cd {PROJECTNAME}
yarn
yarn dev
```

## Features

- Add/Delete Elements
- Duplicate ELements
- Set input required
- Drag elements to reorder
- Multiple element types
  - TextInput
  - TextArea
  - RadioInput
  - NumberInput
  - DateInput
  - TimeInput
